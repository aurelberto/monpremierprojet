Positions et Overflows

Les Positions

Introduction

La propriété CSS position permet de contrôler la manière dont un élément est positionné dans le document.

Elle détermine si un élément suit le flux normal du document ou si il est placé précisément avec les propriétés: top, right, bottom, left.

1. Position: Statique

Comportement : L'élément suit le flux normal, position par défaut de tous les éléments HTML.

top, right, bottom, left n'ont aucun effet si l'élément à cette propriété.

Exemple

        
        .static {
            height: 50px;
            width: 50px;
            background-color: blue;
            position: static;
        }

JE SUIS COLLANT !!

2. Position: relatives

Comportement: l'élément est positionné par rapport à sa position normal

Il occupe toujours sa place dans le flux du document

Les propriétés top, bootom, right et left, le déplace visuellement, sans changer sa place d'origine

        

        .relative { 
    width: 50px;
    height: 50px;
    position: relative;
    background-color: rgb(211, 37, 130);
    top: 2rem;
    left: 3rem;
}

3.Positions absolues

L'élément est retiré du flux normal

Il est positionné par rapport au premier parent (non statique)

Si aucun parent n'est positionné, il se basera sur la balise "html"

4.Positions fixes

L'élément fixé est retiré du flux comme absolute.

Mais il est positionné par rapport à la fenêtre du navigateur.

Il ne bouge pas quand on scroll.

ELEMENT FIXE!!!!!!

5.Positions sticky

Le comportement est un mixte entre un relative et fixed

L'élément reste dans le flux jusqu'à ce qu'on scroll au point où il devient fixe
