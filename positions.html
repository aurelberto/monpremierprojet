Positions et Overflows

Les Positions

Introduction

La propriété CSS position permet de contrôler la manière dont un élément est positionné dans le document.

Elle détermine si un élément suit le flux normal du document ou si il est placé précisément avec les propriétés: top, right, bottom, left.

1. Position: Statique

Comportement : L'élément suit le flux normal, position par défaut de tous les éléments HTML.

top, right, bottom, left n'ont aucun effet si l'élément à cette propriété.

Exemple

        
        .static {
            height: 50px;
            width: 50px;
            background-color: blue;
            position: static;
        }

2. Position Relative

Comportement : L'élément est positionné par rapport à sa position normale.

Il occupe toujours sa place dans le flux du document.

Les propriétés top, right, bottom et left, le déplacent visuellement, sans changer sa place d'origine. 

        
        .relative {
            height: 50px;
            width: 50px;
            background-color: blue;
            position: relative;
            top: 2rem; 
            left: 3rem; 
        }

3. Position Absolue

L'élément est retiré du flux normal

Il est positionné par rapport au premier parent positionné (non statique).

Si aucun parent n'est positionné, il se basera sur la balise "html".
